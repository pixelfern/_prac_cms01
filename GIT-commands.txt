1. git init
2. git status
3. git add . [add all to STAGING]
       add file1.php, file2.php etc
       add *.php [all files with such extension]
4. git commit -m "comment..."
** git commit -am "comment..." [Commits files without Staging]
5. git branch [will list all the brances]
6. git branch **branchName** [will create new branch]
7. git checkout **branchName** [will show you details of that branch]
8. git checkout master [will swich yout to the master branch]
9. git merge **branchName** [will merge brach with the master file]
10. git branch -d **branchName** [will delete that branch]
